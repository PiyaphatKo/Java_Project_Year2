# Vehicle Management GUI (Java Swing)

The **Vehicle Management** program is a Java application developed using **Java Swing** to create a graphical user interface (GUI) for managing vehicle information. It includes menus for adding, editing, and displaying information.

## Main Components
* `web_page`: The main class for calling the GUI.
* `gui_data`: Manages the display of buttons, icons, and panels.
* `update_data`: Manages vehicle information.
* `Show_data`: Displays a dialog for adding, editing, and showing data.
